# DeepfakeNet
This website is a school project for my IT Ethics class. The project is a group project that was worked on by Ewan Milliga, Andrew Boyd, Sean Reichert, and Falah Alfadhli.
# Content
There are 6 pages for each topic and the sources, ranging from the main page to a game simulation page. The information discusses the history of deepfakes, the dangers of deepfakes, the avoidance of deepfakes, and a game to detect deepfakes.
# Details
The target audience for this website is art authenticators, and our "hiring" company is a museum. Our goal is to teach art authenticators about deepfakes and how they can use this information to stop them from entering museums.
